What's included

When you purchase our plugin from codecanyon, you need to download the Easy WooCommerce Quick View Zip files from your codecanyon account. Navigate to your downloads tab on codecanyon and find Easy WooCommerce Quick View. Click the download button.

  • Installable Plugin file only. You can upload this file when you install the Plugin.

  • All files and documentation (full zip folder). You will need to extract and locate the installable WordPress file to upload when you install the plugin.
